1. 根据棋子数目来更改搜索深度

```java
class demo{
    public AlphaBetaNode search(Board board) {
        this.board = board;
        if (board.pieces.size() < 28)
            DEPTH = 3;
        if (board.pieces.size() < 16)
            DEPTH = 4;
        if (board.pieces.size() < 6)
            DEPTH = 5;
        if (board.pieces.size() < 4)
            DEPTH = 6;
    }
}
```
